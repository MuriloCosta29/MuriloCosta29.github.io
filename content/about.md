I'm Murilo Costa, 18, a Computer Science student at CESAR School in Recife, Brazil. My goal is to become a DevOps Engineer before 25.

This blog is where I document the projects I build, the decisions behind them, the bugs I hit, and the tradeoffs I learn to make. Every post comes from something I put my hands on — no theory without practice.

## What I'm building

**Deploy Tracker** — A multi-service monitoring application with FastAPI, PostgreSQL, Redis, Celery, Docker Compose, GitHub Actions, Prometheus, Grafana and Nginx. This is my main project and the subject of most posts on this blog.

**GTD App** — A REST API built with FastAPI, SQLite, SQLAlchemy and Docker. It was my first real backend project and the one that made me take software engineering seriously.

## What I work with

Currently learning: Redis, Celery, CI/CD and observability.
